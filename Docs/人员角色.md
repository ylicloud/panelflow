# 系统中角色:
- 管理员
- 报价员
- 生产管理人员
- 采购人员
- 仓库人员
- 装配人员
- 质检人员

## 管理员: js=0,Admin
- 拥有其他角色的所有权限
- 可以管理所有角色
- 可以管理所有用户

## 报价员: js=2,Quoter
### 报价阶段: 
- 可以查看报价单
- 可以创建报价单
- 可以修改报价单
- 可以删除报价单
- 可以导出报价单
- 可以导入客户BOM表(Excel)
- 可以查看报价单的详细信息
### 签订制造合同阶段:
- 可以查看合同
- 可以创建合同
- 可以修改合同
- 可以删除合同
- 可以打印合同

## 生产管理人员: js=6,ProjectManager
### 安排生产阶段:
- 可以查看生产任务单
- 可以打印生产任务单
- 可以下达采购计划
- 可以创建装配任务单
- 可以创建质检任务单
- 可以指定装配小组
- 可以指定质检人员
### 发货阶段:
- 可以创建发货单
- 可以填写装箱单
- 可以录入发运记录

## 采购人员: js=3,Purchaser
### 采购元件阶段:
- 可以查看采购计划
- 可以录入采购进度

## 仓库人员: js=7,Warehouse
### 入库阶段:
- 可以查看库存
- 可以填写到货验收单
- 可以填写入库单
- 可以填写报销单
- 可以发起盘点
### 出库阶段:
- 可以填写出库单（领料）
- 可以填写出库单（发货）

## 装配人员: js=5,Assembler
### 装配阶段:
- 可以查看装配任务
- 可以录入装配进度

## 质检人员: js=4,QualityInspector
### 质量检验阶段:
- 可以查看质检任务
- 可以录入质检进度
- 可以打印检验记录
- 可以打印装配卡（合格证）
- 可以填写不合格品处理单
