# 壳体类型与材质分类
> **文档来源**：手写笔记识别整理（图片：控制柜柜体类型与材质手稿）。
> **用途**：作为"柜体模板（Cabinet Template）"独立功能的**基础字典数据**，与现有 PB 程序无关，不影响原有数据库。
> **状态**：草稿 v0.1.



## 一、识别原则与命名约定

- **壳体类型（CabinetType）**：表示壳体的整体形态，共有5种: `柜`、`台`、`箱`、`盒`、`其他`。
  其中,"柜"表示控制柜, "台"表示操作台, "箱"表示操作箱, "盒"表示操作盒, "其他"表示其他类型.
- **材质（Material）**：表示构成柜体某一部位（面板、内门、台面、外门）的具体材料或表面工艺，例如 `不锈钢`、`镀锌板`、`喷塑涂层镀锌板`。
- **部位（Part）**：在一种柜体类型内部，材质所归属的位置，例如 `外门`、`内门`、`台面`、`面板`、`支架`。
- 一种柜体类型由"多个部位 × 各自材质"组合而成，这正是后续让用户"自由拼装柜体模板"的字典基础。

--- 五. 需求
  1. 用户在导入报价单,或填写报价单时,需要选择壳体类型,并填写壳体相关信息.
  2. 为了简化用户操作,独立增加一个功能,用于管理壳体类型与材质.
