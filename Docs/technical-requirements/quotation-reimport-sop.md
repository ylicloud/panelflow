# 报价单重新导入 Excel 元件清单操作指引（不改代码）

## 1. 标准操作路径（已确认）

当前系统支持在“空报价单”和“非空报价单”上执行导入保存。

- `dqzt=1`（空报价单）：点击保存后直接执行保存流程
- `dqzt!=1`（非空报价单）：点击保存时弹出两次确认，确认后覆盖保存

## 2. 业务用户 4 步 SOP

适用对象：需要替换为“新 Excel 元件清单”的业务用户。

### 步骤 1：进入目标报价单的“导入元件”页

- 在报价单列表点击“导入excel元件”图标进入页面

### 步骤 2：导入 Excel

- 点击“打开excel元件表”
- 选择新的 `.xls` / `.xlsx` 文件
- 页面提示“已加载 N 行数据”后进入下一步

### 步骤 3：校验并预览

- 点击“数据检查”，需显示“数据检查通过”
- 点击“目录预览”，需显示“已生成 N 个目录子节点”
- 如中途改动表格（改单元格/增删行），必须重新执行“数据检查 + 目录预览”

### 步骤 4：保存方案

- 确认左侧“保存方案”按钮可点击后执行保存
- 成功提示应为“保存成功，共写入 X 条记录”
- 若当前为非空报价单（`dqzt!=1`），系统会连续弹出两次确认：
  1. `当前报价单有数据,导入时会清空原来数据,是否继续?`
  2. `确定要覆盖当前报价单吗?`
- 两次都确认后执行覆盖保存

## 3. 风险控制建议

- 非空报价单覆盖保存会先清空原有 `BJB` 明细再重建，请先确认业务影响
- 建议在覆盖前先导出/备份当前数据，便于回溯
- 若为关键报价单，建议由报价负责人复核后再执行覆盖

## 4. 风险提示

- 覆盖保存属于高影响操作，请谨慎确认两次弹窗内容
- 覆盖后如需还原，需依赖事前备份数据
