# 报价单重新导入 Excel 元件清单操作指引

## 1. 标准操作路径（已确认）

当前系统支持在“空报价单”和“非空报价单”上执行导入保存，并支持 **覆盖** 与 **追加**。

- `dqzt=1`（空报价单）：目录预览无需选择模式，直接按覆盖预览；保存时不再二次确认
- `dqzt!=1`（非空报价单）：目录预览时选择「覆盖 / 追加 / 取消」；保存时按所选模式再确认一次

## 2. 业务用户 SOP

适用对象：需要用新 Excel 元件清单更新或扩展报价单的业务用户。

### 步骤 1：进入目标报价单的“导入元件”页

- 在报价单列表点击“导入excel元件”图标进入页面

### 步骤 2：导入 Excel

- 点击“打开 Excel 元件表”
- 选择新的 `.xls` / `.xlsx` 文件
- 页面提示“已加载 N 行数据”后进入下一步

### 步骤 3：校验并预览

- 点击“数据检查”，需显示“数据检查通过”
- 点击“目录预览”：
  - **空单**：直接生成目录节点
  - **非空单**：选择其一
    - **覆盖原报价单**：树中仅显示本次 Excel 的柜
    - **追加到原报价单**：树中保留原柜，新增柜带「新增」标记
    - **取消**：树还原，保存方案不可用
- 如中途改动表格（改单元格/增删行），必须重新执行“数据检查 + 目录预览”

### 步骤 4：保存方案

- 确认「保存方案」可点击后执行
- 覆盖：确认后整单重建（原柜编码从 0001 重排）
- 追加：确认后保留原柜，新柜从最大柜码续编
- 成功提示含单元数 / 元件数；保存时可见进度 `(xxx/yyy)`

## 3. 两类 Excel 导出

| 按钮 | 数据源 | 用途 |
|------|--------|------|
| 另存表格 | 当前页面表格 | 改表后的临时备份（与覆盖/追加无关） |
| 导出方案 Excel | 数据库 BJB | 整单复用；可再「打开 Excel」导入其他报价单 |

## 4. 风险控制建议

- **覆盖**会先清空原有 `BJB` 明细再重建，请先确认业务影响；建议先「导出方案 Excel」备份
- **追加**不改原柜编码与价格，仅新增柜；适合在已有方案上扩展
- 若为关键报价单，建议由报价负责人复核后再执行覆盖

## 5. 风险提示

- 覆盖属于高影响操作，请看清目录预览所选模式与保存确认文案
- 覆盖后如需还原，需依赖事前备份（导出方案 Excel）或数据库备份
